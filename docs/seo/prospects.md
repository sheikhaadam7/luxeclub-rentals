# Backlink Prospect List

**Last updated:** 2026-04-11

## How to use this list

1. Start with **Priority 1** outlets in each tier. These are the ones where a single placement moves the needle most on either SEO authority or advance-booker audience reach.
2. Run in parallel with `content-calendar.md` — every assignment goes in the pitch queue there.
3. The **DR band** is a rough estimate from public signals (Alexa rank residue, inbound-link patterns, public reputation) since we don't have an Ahrefs seat. High = 80+, Medium = 60–80, Low = <60. Always verify if the decision hinges on it.
4. The **audience fit** column matters more than DR for this business — a High-fit/Medium-DR outlet beats a Low-fit/High-DR outlet every time because the latter won't convert.
5. **Contact URL** is a starting point for your insider; they'll have better internal contacts than these public pages.

## Notes on Ahrefs "broken JS" type findings

Some of these outlets (particularly the European majors) will run their own crawler checks on linked sites. Ensure the target URL returns HTTP 200 and renders without console errors before the insider submits. `content-calendar.md` has a pre-pitch URL validity check as part of its monthly routine.
