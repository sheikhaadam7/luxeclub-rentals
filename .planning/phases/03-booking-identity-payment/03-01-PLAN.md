1. Create `lib/pricing/calculator.ts`:
   - Export `BookingPriceBreakdown` interface with fields: `baseRate`, `rentalDays`, `rentalSubtotal`, `deliveryFee`, `returnFee`, `depositAmount`, `noDepositSurcharge`, `totalDue`
   - Export `calculateBookingTotal(vehicle, formValues)` pure function:
     - `vehicle` parameter: `Pick<Vehicle, 'daily_rate' | 'weekly_rate' | 'monthly_rate' | 'deposit_amount'>` — the deposit_amount column is added to vehicles by the migration in Task 1
     - `formValues` parameter: `{ startDate: Date, endDate: Date, durationType: 'daily' | 'weekly' | 'monthly', pickupMethod: 'delivery' | 'self_pickup', returnMethod: 'collection' | 'self_dropoff', depositChoice: 'deposit' | 'no_deposit' }`
   - Calculation logic:
     - `rentalDays = differenceInDays(endDate, startDate) + 1` (inclusive)
     - For daily: `baseRate = vehicle.daily_rate`, `rentalSubtotal = baseRate * rentalDays`
     - For weekly: `baseRate = vehicle.weekly_rate ?? vehicle.daily_rate * 7`, `rentalSubtotal = baseRate * Math.ceil(rentalDays / 7)`
     - For monthly: `baseRate = vehicle.monthly_rate ?? vehicle.daily_rate * 30`, `rentalSubtotal = baseRate * Math.ceil(rentalDays / 30)`
     - `deliveryFee = pickupMethod === 'delivery' ? 50 : 0`
     - `returnFee = returnMethod === 'collection' ? 50 : 0`
     - `depositAmount = depositChoice === 'deposit' ? (vehicle.deposit_amount ?? 5000) : 0`
     - `noDepositSurcharge = depositChoice === 'no_deposit' ? (vehicle.daily_rate * 0.30 * rentalDays) : 0`
     - `totalDue = rentalSubtotal + deliveryFee + returnFee + noDepositSurcharge` (deposit is authorized separately, not included in totalDue)
   - All number operations use `Math.round()` to 2 decimal places where needed
