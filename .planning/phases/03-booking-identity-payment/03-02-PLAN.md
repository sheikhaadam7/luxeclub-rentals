   - Optional start time input (HH:MM format, default "10:00")
   - Wire all fields to form via `form.register()` and `form.setValue()` for DayPicker (DayPicker doesn't support register directly — use `onSelect` to call `form.setValue('startDate', range.from)` and `form.setValue('endDate', range.to)`)

     - `<AddressAutofill accessToken={process.env.NEXT_PUBLIC_MAPBOX_TOKEN!}>` wrapping a text input
     - On address selection (`onRetrieve`), extract coordinates and call `form.setValue('deliveryLat', lat)`, `form.setValue('deliveryLng', lng)`, `form.setValue('deliveryAddress', fullAddress)`
     - Show `<AddressMinimap>` below with draggable pin for pinpoint confirmation, `canAdjustMarker` and `satelliteToggle` enabled

   - Return method section: two cards "Self-Drop-Off (Free)" and "Collection (AED 50)" — radio-style toggle
