**2. Create `app/actions/admin.ts`** â€” Server Actions for admin operations:

```typescript
'use server'
import { createClient } from '@/lib/supabase/server'
import { createAdminClient } from '@/lib/supabase/admin'
```

Implement two Server Actions:

**a. `updateVehicleAvailability(vehicleId: string, isAvailable: boolean, overrideNotes: string)`:**
- First verify caller is admin: `const supabase = await createClient()` then `const { data: claimsData } = await supabase.auth.getClaims()` then check `claims.sub` against profiles table: `supabase.from('profiles').select('role').eq('id', claims.sub).single()` â€” if role !== 'admin', return `{ error: 'Unauthorized' }`
- Use admin client for the write: `const admin = createAdminClient()` then `admin.from('vehicles').update({ is_available: isAvailable, override_notes: overrideNotes, updated_at: new Date().toISOString() }).eq('id', vehicleId)`
- Return `{ error: null }` on success or `{ error: message }` on failure

**b. `getAdminDashboardData()`:**
- Verify admin role (same pattern as above)
- Query latest scraper_run: `supabase.from('scraper_runs').select('*').order('ran_at', { ascending: false }).limit(1).single()`
- Query all vehicles: `admin.from('vehicles').select('id, slug, name, is_available, override_notes, scraped_at').order('name')`
- Compute staleness: `isStale = !lastRun || new Date(lastRun.ran_at) < new Date(Date.now() - 24 * 60 * 60 * 1000)`
- Return `{ lastRun, vehicles, isStale }`
